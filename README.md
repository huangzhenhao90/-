代码完全由 cursor 生成。包含以下功能。
1. 使用自己的 api key 和 base_url。
2. 总结网页内容，生成一句话总结、摘要、核心观点、金句提取
3. 跟网页内容进行对话，提问。

如何使用：
1. 下载代码，解压后，把文件夹拖到 chrome 浏览器的「管理扩展程序」界面（先打开开发者模式）
2. 找到插件 logo，右键「选项」，填入你的api key 和 base_url
3. 到你想要总结的网页，点击插件 logo，就会自动生成相应总结。
4. 在输入框里输入你的问题，跟网页对话。
